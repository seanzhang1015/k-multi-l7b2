# K模块 Multi计算器 L7B2 - 使用说明

**在线地址**: https://seanzhang1015.github.io/k-multi-l7b2/

## 功能说明

L7B2策略仓位倍数计算器，输入股票代码、收盘价、买入日星期，即可计算出 multi 值并判断是否可买入。

---

## 安装到手机（PWA方式，推荐）

PWA 支持离线使用，安装后像原生 App 一样运行，Android 和 iOS 通用。

### 前置步骤：启动本地服务器

PWA 要求通过 HTTP 访问（不能直接打开文件），需要先启动一个本地服务器：

1. 电脑上打开命令行，进入 k_mult_app 目录：
   ```
   cd D:\Abacktesting\Gen26_backtesting_2diverges_win\Gen26\k_mult_app
   ```
2. 启动服务器：
   ```
   python -m http.server 8080
   ```
3. 电脑浏览器访问 `http://localhost:8080` 确认能打开

### 手机安装

确保手机和电脑在**同一 WiFi 网络**下。

1. 查看电脑局域网 IP：
   - Windows: 命令行运行 `ipconfig`，找到"IPv4 地址"（通常 192.168.x.x）
2. 手机浏览器访问 `http://电脑IP:8080`

#### Android（Chrome 浏览器）
1. 打开页面后，Chrome 底部会弹出"添加到主屏幕"提示，点击即可
2. 或点击浏览器菜单 (⋮) → "添加到主屏幕"
3. 桌面出现 K-Multi 图标，点击即可使用

#### iOS（Safari 浏览器，仅 Safari 支持）
1. 打开页面后，点击底部分享按钮 (⬆️)
2. 向下滑动，点击"添加到主屏幕"
3. 点击"添加"，桌面出现 K-Multi 图标

### 安装后可离线使用

首次加载后所有资源被缓存，之后即使断网也能正常使用计算器。

## 输入参数

| 参数 | 说明 | 示例 |
|------|------|------|
| 股票代码 | 6位数字 | 300001、001001、600000 |
| 涨停日收盘价 | 涨停当天收盘价（元） | 45 |
| 买入日星期 | 实际买入日是周几 | 周三 |

---

## L7B2 策略参数

```
multi = base × 市场权重 × 价格权重 × 星期权重 × 前缀权重
```

### 市场权重
- 创业板 (30开头): 50.0
- 其他: 1.0

### 价格权重
| 价格区间 | 权重 |
|---------|------|
| 50元以上 | 0.8 |
| 40-50元 | 5.0 |
| 30-40元 | 10.0 ★ 最优 |
| 20-30元 | 0.6 |
| 10-20元 | 1.0 |
| 5-10元 | 0.4 |
| 5元以下 | 0.5 |

### 星期权重
| 星期 | 权重 |
|------|------|
| 周一 | 0.9 |
| 周二 | 1.0 |
| 周三 | 1.2 ★ 最优 |
| 周四 | 0.05 |
| 周五 | 0.85 |

### 前缀权重
| 前缀 | 权重 |
|------|------|
| 001 | 120.0 ★ 最优 |
| 300 | 10.0 |
| 603 | 2.0 |
| 601 | 2.0 |
| 002 | 1.18 |
| 000 | 1.0 |
| 605 | 0.3 |
| 600 | 0.05 |
| 003 | 0.05 |

### 限制
- 最大 multi 上限: 15.0

---

## 判断标准

| multi 值 | 判定 | 操作 |
|----------|------|------|
| multi < 0.5 | 不可买入 | 低于阈值 |
| 0.5 ≤ multi < 0.65 | 可买入 | 低质量区 |
| 0.65 ≤ multi < 1.3 | **死区跳过** | 亏损区，不买入 |
| multi ≥ 1.3 | 可买入 | 黄金区 |

**死区说明**: mult 在 0.65~1.3 范围内的交易胜率仅 56.8%，是策略亏损的重灾区，必须跳过。

## 计算示例

### 示例1: 001开头创业板
- 股票: 001001
- 收盘价: 35元
- 买入日: 周三

计算: 0.5 × 1.0(非30开头) × 10.0(30-40元) × 1.2(周三) × 120.0(001) = **720.0** → 限制为 **15.0** (黄金区，可买入)

### 示例2: 创业板中价股
- 股票: 300001
- 收盘价: 25元
- 买入日: 周二

计算: 0.5 × 50.0(创业板) × 0.6(20-30元) × 1.0(周二) × 10.0(300) = **150.0** → 限制为 **15.0** (黄金区，可买入)

### 示例3: 死区示例
- 股票: 000001
- 收盘价: 8元
- 买入日: 周五

计算: 0.5 × 1.0 × 0.4(5-10元) × 0.85(周五) × 1.0(000) = **0.17** (低于阈值，不可买入)

### 示例4: 上海主板高价股
- 股票: 601001
- 收盘价: 55元
- 买入日: 周四

计算: 0.5 × 1.0 × 0.8(50元以上) × 0.05(周四) × 2.0(601) = **0.04** (不可买入)
